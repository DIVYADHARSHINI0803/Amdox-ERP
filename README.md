# High-Performance Search Interface (React + TypeScript)

This project includes both frontend and backend implementations.

## 📁 Project Structure
- frontend/ → React + TypeScript app
- backend/ → API services (if applicable)

## 🚀 Getting Started

### Frontend
Go to the frontend folder:
cd frontend

Install dependencies:
npm install

Run the app:
npm start

### Backend
Go to backend folder and follow setup instructions (if available).

## 📌 Features
- Custom Hooks (useLocalStorage, useDebounce)
- Reusable Button Component
- Optimized Search Interface
